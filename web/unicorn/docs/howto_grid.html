{{ page.title }}

Defined in _base.scss with these settings: 

$total-columns  : 12;
$column-width   : 4em;
$gutter-width   : 1em;
$grid-padding   : $gutter-width;

$container-style: fluid;

The core defines a row container class, and various span classes that are placed in the row to build up the grid:

span1
span6
span3
span3
span6
span6
span8
span4
span1
span3 push3
span5

The span elements float to the left, except for the last span, which is always floated to the right.

You can leverage Sass to construct more semantic class names, if that's your preference: 

	// bookcover is 1/12 columns wide
 	.bookcover {
 		@include span-columns(1,12);
 	}
