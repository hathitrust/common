This glyph set can be extended as necessary.

Adding an icon to a link

There are several ways to use the icon glyphs. The core defines a Sass module that manages all the details of using one of the glyphs: 

    .endnotelink{
      @include link-icon(";");
      margin-right: 1em;
      padding:.5em;
    }

Or, you can add the class supplied by IcoMoon: 

<a class="endnotelink icomoon-download" href="...">Export EndNote</a>

Without using the Sass module and/or extending the element's classes, you'd have to duplicate the full CSS with every element: 

    .endnotelink {
        content: "; ";
        font-family: 'icomoon';
        font-style: normal;
        speak: none;
        font-weight: normal;
        line-height: 1;
        -webkit-font-smoothing: antialiased;
        margin-right: 2px;
        color: #666;
        display: inline-block;
        font-size: 110%;
    }
